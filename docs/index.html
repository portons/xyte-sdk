Quick Start

Run xyte, paste your API key, then access your dashboard. For automation, pass --tenant and use JSON outputs.

Safety by Default

Read-only by default. Non-read endpoint calls require --allow-write. Destructive calls require --confirm <endpoint-key>.

Headless for Agents

Use xyte tui --headless --format json and parse NDJSON runtime frames with contract xyte.headless.frame.v1.

Reporting

Generate branded PDF findings with xyte report generate. Default output is redacted unless --include-sensitive is used.

End-to-End Command Flow

npm install
npm run build
xyte
xyte setup status --tenant acme-demo --format json
xyte inspect deep-dive --tenant acme-demo --window 24 --format json > /tmp/deep-dive.json
xyte report generate --tenant acme-demo --input /tmp/deep-dive.json --out /tmp/xyte-findings.pdf
xyte tui --headless --screen dashboard --format json --once --tenant acme-demo
xyte mcp serve
